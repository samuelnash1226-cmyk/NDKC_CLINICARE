## 🐛 Troubleshooting

### If you still see permission errors:

1. **Verify rules deployed:**
   - Go to Firebase Console → Firestore → Rules
   - Check "Last deployment" timestamp
   - Should be recent (just now)

2. **Check user role:**
   - Go to Firestore → users collection
   - Find your user document
   - Verify `role` field is set to `admin` or `nurse`

3. **Clear cache:**
   - Close all browser tabs
   - Clear browser cache
   - Re-login to app

4. **Check console:**
   - Press F12 (Developer Tools)
   - Look for red errors
   - Share error messages if issue persists
