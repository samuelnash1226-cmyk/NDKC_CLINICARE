# 🚀 Quick Reference Card - Medical Certificates & Offline Mode

## 📋 For Nurses: How to Generate a Medical Certificate

### Step-by-Step Guide

1. **Open Nurse Dashboard**
   - Click "Add Visit" button (green button at top)

2. **Fill Out Visit Form**
   - Select grade level first
   - Choose student from list
   - Enter symptoms
   - Enter treatment provided
   - Add any notes (optional)
   - Toggle parent notification if needed
   - Check "needs pickup" if applicable

3. **Submit the Visit**
   - Click "Log Visit" button
   - Wait for success confirmation (green checkmark)

4. **Certificate Appears Automatically** 🎉
   - After 1.5 seconds, certificate modal opens automatically
   - No need to click anything!

5. **Select Certificate Type**
   - **Clinic Visit Proof** - Standard receipt (recommended)
   - **Medical Certificate** - Formal medical document
   - **Excuse Slip** - For class absence

6. **Download or Print**
   - Click "Download as PDF" for official records
   - Click "Download as PNG" for digital sharing
   - Click "Download as JPEG" for smaller file size
   - Click "Print Certificate" to print directly

7. **Done!**
   - Click the X to close and return to dashboard

## 🌐 Offline Mode Indicators

### What You'll See

**🟢 Online (Normal)**
- Bottom right: Green wifi icon + "Online"
- Last sync time shown
- All features available

**🟡 Going Offline**
- Orange banner appears at top
- "You're Offline" message
- Some features limited

**🟢 Back Online**
- Green banner appears at top
- "Back Online!" message
- Auto-syncs data
- Disappears after 3 seconds

### What Works Offline?
- ✅ View existing data (cached)
- ✅ Browse dashboards
- ✅ View visit history (cached)
- ❌ Cannot log new visits
- ❌ Cannot send notifications
- ❌ Cannot create new users

### What Happens When Back Online?
- ✅ Data automatically syncs
- ✅ All features restored
- ✅ Any queued actions processed

## 🔍 Troubleshooting

### Certificate doesn't appear after logging visit
**Solution:** Wait 1.5 seconds. If still doesn't appear, refresh page and try again.

### PDF won't download
**Solution:** Check browser pop-up blocker. Allow downloads from this site.

### Print looks wrong
**Solution:** Make sure "Background Graphics" is enabled in print settings.

### Offline banner won't go away
**Solution:** Click the X button to dismiss. Or wait until back online.

### Image quality is poor
**Solution:** Use PDF format instead. It has the best quality.

### Can't see campus image on login
**Solution:** You're on mobile. Campus image only shows on desktop/tablet.

## ❓ FAQ

**Q: Can I edit a certificate after it's generated?**  
A: No. Certificates are final. To make changes, submit a new visit with corrections.

**Q: How long are certificates stored?**  
A: Forever in Firestore database. Downloads are stored on your device.

**Q: Can parents access certificates?**  
A: Currently no. Future update will add this feature.

**Q: What if I close the certificate modal by accident?**  
A: Go to Visit History and view the visit record. You can regenerate (future feature).

**Q: Are certificates legally valid?**  
A: Yes, they include all required information and nurse signature.

**Q: Can I customize the certificate design?**  
A: Admins can modify the template in code. Contact IT team.

**Q: What format is best for emailing?**  
A: PDF for official, PNG for quick sharing, JPEG for smaller size.

**Q: Does the certificate include test results?**  
A: Only if added in visit notes. Otherwise, attach separately.
