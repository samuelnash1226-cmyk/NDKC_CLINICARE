## 🆘 TROUBLESHOOTING:

### No SMS received?

1. **Check phone number format:**
   - Should be: `09516495356`
   - NOT: `+639516495356` (system auto-formats)

2. **Verify setup completed:**
   - Re-run `setup-fmcsms.html`
   - Look for "Success!" message

3. **Check console logs:**
   - Press F12 in browser
   - Look for: `✅ SMS sent successfully`

4. **Test API directly:**
   - Open `test-fmcsms.html`
   - Send test SMS to your phone

### Still not working?

1. **Check FMCSMS dashboard:**
   - Login to https://fortmed.org/web/FMCSMS/
   - Verify API key is active
   - Check account credits

2. **Read full guide:**
   - Open `FMCSMS_SETUP.md`
   - Complete troubleshooting section
